# angiogenese_tumorale
## Modélisation de la croissance tumorale avec angiogenèse et diffusion du VEGF : Développement et simulation d’un modèle mathématique spatio-temporel basé sur des équations aux dérivées partielles pour simuler la progression tumorale et l'impact de la chimiothérapie. Utilisation de Python pour les simulations et analyses des interactions cellulaires
